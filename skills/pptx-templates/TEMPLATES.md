# PPT Templates Library

基于 Anthropic PPTX Skill 最佳实践和专业设计原则构建的高质量模板库。

## 模板分类

### 1. 商务演示模板

#### 1.1 Executive Report (高管报告)
- **配色:** Midnight Executive (深蓝 + 冰蓝 + 白)
- **适用场景:** 季度报告、年度总结、董事会汇报
- **特点:** 专业、稳重、数据驱动
- **页面结构:**
  - 封面：深色背景 + 侧边强调条
  - 目录：4 宫格卡片布局
  - 内容页：左侧导航 + 右侧内容
  - 数据页：大数字卡片 + 趋势图表
  - 封底：深色背景 + 关键信息摘要

#### 1.2 Startup Pitch (创业融资)
- **配色:** Coral Energy (珊瑚红 + 金色 + 深蓝)
- **适用场景:** 融资路演、产品发布、投资人演示
- **特点:** 活力、创新、视觉冲击
- **页面结构:**
  - 封面：全屏背景图 + 大字标题
  - 问题/解决方案：对比布局
  - 市场规模：大数字 + 增长箭头
  - 商业模式：流程图/信息图
  - 团队介绍：头像 + 简介卡片
  - 融资需求：清晰的数字展示

#### 1.3 Product Launch (产品发布)
- **配色:** Ocean Gradient (深海蓝 + 青色 + 午夜蓝)
- **适用场景:** 新产品发布、功能介绍、技术演示
- **特点:** 科技感、现代、简洁
- **页面结构:**
  - 封面：产品大图 + 渐变背景
  - 痛点分析：图标 + 简短描述
  - 解决方案：产品截图/渲染图
  - 核心功能：网格布局 + 图标
  - 技术优势：对比表格
  - 定价/发布：大号数字 + 日期

### 2. 技术演示模板

- **适用场景:** API 介绍、开发者文档、集成指南

## 通用设计组件

### 封面页变体

```javascript
// 1. 经典封面 (深色背景)
background: { color: COLORS.dark }
左侧强调条：RECTANGLE (0.3" 宽，全高，accent 色)
标题：44pt, bold, 白色
副标题：18pt, secondary 色
日期/保密：12pt, muted

// 2. 现代封面 (浅色背景 + 几何图形)
background: { color: 'FFFFFF' }
装饰图形：大圆角矩形 (accent 色，透明度 10%)
标题：44pt, bold, primary 色
副标题：18pt, muted 色

// 3. 图片封面 (全屏背景)
background: { path: 'hero-image.jpg' }
遮罩：半透明深色矩形 (透明度 40%)
标题：44pt, bold, 白色
副标题：18pt, 白色
```

### 目录页布局

```javascript
// 4 宫格卡片
cards = [
  { title: '章节 1', number: '01', color: COLORS.primary },
  { title: '章节 2', number: '02', color: COLORS.success },
  { title: '章节 3', number: '03', color: '#3B82F6' },
  { title: '章节 4', number: '04', color: '#8B5CF6' }
]
布局：2x2 网格，每张卡片 4" x 2"
```

### 过渡页设计

```javascript
// 章节过渡页
background: { color: COLORS.primary }
章节编号：72pt, bold, 白色，透明度 30%
章节标题：36pt, bold, 白色
章节副标题：16pt, secondary 色
```

### 数据展示组件

```javascript
// 大数字卡片
card: RECTANGLE (2.1" x 1.2", 白色背景，边框)
图标/emoji: 20pt
数字：22pt, bold, primary 色
标签：9pt, muted 色
变化：10pt, success 色 (正) / warning 色 (负)

// 统计行
4 个卡片水平排列，间距 0.25"
```

### 图表样式规范

```javascript
// 柱状图
chartColors: [COLORS.primary, COLORS.success, '#3B82F6']
showValue: true
dataLabelPosition: 'outEnd'
valGridLine: { color: 'E5E7EB', size: 0.5 }

// 饼图
chartColors: [COLORS.primary, '#6B7280', '#3B82F6', COLORS.success]
showPercent: true

// 折线图
lineSize: 3
lineSmooth: true
chartColors: [COLORS.success]
showValue: true
```

```javascript
// 应用到幻灯片
const COLORS = template;
slide.background = { color: COLORS.primary };
slide.addText('标题', { color: COLORS.accent });
```

## 最佳实践检查清单

生成 PPT 前检查：
- [ ] 选择了与内容匹配的配色方案
- [ ] 确定了主导色（60-70% 视觉权重）
- [ ] 规划了视觉元素（每页至少 1 个）
- [ ] 避免了重复布局
- [ ] 设置了合适的字体大小对比
- [ ] 留出了足够的边距和呼吸空间
- [ ] 确保了文字与背景的对比度
- [ ] 避免了装饰性标题下划线

生成后 QA：
- [ ] 提取内容检查（markitdown）
- [ ] 生成缩略图视觉检查
- [ ] 检查拼写和数字准确性
- [ ] 验证图表数据正确性
- [ ] 确认页码和页脚一致
